Related Doc: package rete

case class History(fact: Map[Symbol, Any], eventNumber: Int, event: Map[Symbol, Any], rule: String, histories: List[History], message: String) extends Product with Serializable

This class represents the history of a created fact. The history outlines what event lead to the creation of the fact, what rule created it; and finally other relevant events, rules fired, and facts created (the cone of influence).

fact

the generated fact.

eventNumber

position of triggering event in input trace.

event

the event causing the rule to fire.

rule

the rule that fired.

histories

the histories of the previous events, rules, and facts.

message

an error message in case the history was created due to a failure.
